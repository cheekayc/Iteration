## *Lists*

``` r
vec_numeric = 5:8
vec_char = c("My", "name", "is", "Jeff")
vec_logical = c(TRUE, TRUE, TRUE, FALSE)
```

## *`for` loops*

``` r
input = list(...)
output = list(...)
 for (i in 1:n) {
   output[[i]] = f(input[[i]])
 }
```
